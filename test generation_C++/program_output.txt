**Phương pháp lập trình**
  - Khai báo mảng ký tự `s[1010]` để lưu chuỗi.  
  - Sử dụng `scanf("%[^\n]", s)` để đọc toàn bộ một dòng cho đến khi gặp ký tự xuống dòng (`\n`).
  - In chuỗi vừa nhập kèm dấu ngoặc kép bằng `printf("\"%s\"", s)`.  
    - `\"` in ra dấu ngoặc kép.  
    - `%s` in ra nội dung chuỗi. 
  
**Ý nghĩa của "scanset" `%[...]`**:  
  - `%[...]` là định dạng đọc một chuỗi gồm các ký tự thuộc tập ký tự chỉ định.
  - Với `%[^\n]`, dấu `^` nghĩa là "ngoại trừ", nên nó đọc tất cả các ký tự trừ ký tự xuống dòng `\n`.

Lời giải lập trình bằng C, sử dụng `scanf` với format string `[^\n]` để nhập xâu:
```c
#include <stdio.h>

int main() {
   char s[1010];
   scanf("%[^\n]", s);
   printf("\"%s\"", s);
   return 0;
}
```

Lời giải lập trình bằng C, sử dụng `fgets` để nhập xâu:

```c
#include <stdio.h>
#include <string.h>

int main() {
   char s[1010];
   fgets(s, sizeof(s), stdin);
   size_t len = strlen(s);
   if (len > 0 && s[len - 1] == '\n') {
      s[len - 1] = '\0';
   }
   printf("\"%s\"", s);
   return 0;
}
```

Lời giải lập trình bằng C++, sử dụng `getline` để nhập xâu:

```cpp
#include <iostream>
#include <string>

int main() {
   std::string s;
   std::getline(std::cin, s);

   std::cout << "\"" << s << "\"";

   return 0;
}
```
